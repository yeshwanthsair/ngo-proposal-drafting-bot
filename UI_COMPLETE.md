## 🎨 Visual Effects

### Gradients
- **Angle**: 135 degrees (modern diagonal)
- **Smoothness**: Linear transitions
- **Usage**: Headers, buttons, backgrounds

### Shadows
- **Card Shadow**: `0 2px 8px rgba(0, 0, 0, 0.05)`
- **Hover Shadow**: `0 4px 12px rgba(color, 0.3)`
- **Depth**: Subtle, not overwhelming

### Transitions
- **Duration**: 0.3s ease
- **Properties**: All interactive elements
- **Effects**: Smooth color, shadow, transform changes

### Borders
- **Radius**: 8-12px (rounded corners)
- **Accent**: 4px left border on status boxes
- **Color**: Light gray (#E0E0E0)

## 🚀 How to Use

### 1. Start Backend
```bash
python -m uvicorn backend.main:app --host 0.0.0.0 --port 8000 --reload
```

### 2. Start Frontend
```bash
streamlit run frontend/app.py
```

### 3. Open Browser
```
http://localhost:8501
```

## 🔧 Customization

### Change Primary Color
1. Edit `frontend/app.py` line 48: `--primary: #2E7D32;`
2. Update `.streamlit/config.toml`: `primaryColor = "#2E7D32"`

### Change Gradient
1. Edit page header CSS in `frontend/app.py`
2. Modify `linear-gradient(135deg, color1 0%, color2 100%)`

### Change Theme
1. Edit `.streamlit/config.toml`
2. Update all color variables
